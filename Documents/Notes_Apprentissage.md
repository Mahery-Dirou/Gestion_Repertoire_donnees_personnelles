# Projet de gestion d'un repertoire de données personnelles

Le but de ce projet est de se familiariser avec le terminal de Linux.

Ce document résume les commandes et notions apprise durant ce projet.

Les commandes utilisées sont:
	- ls, cd, pwd: naviguer et vérifier l'emplacement.
	- mkdir: créer des sous-dossiers.
	- touch: créer un fichier vide.
	- cp: copier un fichier ou un dossier.
	- mv: déplacer our renommer un fichier/dossier.
	- rm: supprimer un fichier.
	- cat, head, tail: lire le contenu d'un fichier.
	- nano/vim: éditer un fichier
